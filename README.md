# University Portfolio

Personal portfolio website showcasing projects, tech stack, and university progress.

**Live:** [fabian-vargas.dev](https://fabian-vargas.dev)

## Tech Stack

- Astro
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
